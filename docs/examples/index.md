# 示例模板导览

示例按“可直接套用”的业务场景组织。每个示例都包含：

- 场景说明
- 最小 schema
- 关键点

## 基础示例

- [基础表单](/examples/basic-form)：新增/编辑表单
- [表格 CRUD](/examples/table-crud)：完整列表页
- [表单与详情复用](/examples/detail-reuse)：编辑和查看共用 schema
- [自定义扩展组件](/examples/custom-component)：接入项目组件

## SuperTable 模板

- [基础列表](/examples/table-template-basic-list)：只有查询和展示
- [弹窗 CRUD](/examples/table-template-modal-crud)：新增、编辑、删除、详情
- [列表 + 详情](/examples/table-template-master-detail)：强调查看明细
- [行内编辑](/examples/table-template-inline-edit)：轻量行内维护
