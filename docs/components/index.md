# 组件导览

组件文档用于回答“页面里应该用哪个组件或 hook”。

## 核心入口

- [SuperTable](/components/super-table)：列表页核心入口，整合表格、搜索表单、编辑表单、详情和按钮
- [useTable](/components/use-table)：注册并控制 `SuperTable`

## 表单与详情

- [SuperForm](/components/super-form)：配置化表单
- [useForm](/components/use-form)：注册并控制表单
- [SuperDetail](/components/super-detail)：查看态详情展示
- [useDetail](/components/use-detail)：注册并控制详情数据

## 辅助能力

- [SuperButtons / useButtons](/components/buttons)：按钮组
- [useModal / useModalForm](/components/modal)：弹窗与弹窗表单
