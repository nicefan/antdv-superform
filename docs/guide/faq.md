# 常见问题

## 1. 这个库适合什么项目

适合字段多、结构稳定、重复表单和列表较多的中后台项目。

## 2. 详情页一定要单独定义 schema 吗

不一定。更推荐复用表单 schema，再通过查看态能力控制展示。

## 3. 搜索区和编辑表单要不要共用字段定义

可以复用思路，但不建议强行完全共用。搜索区通常只保留高频字段，编辑表单会更完整。

## 4. 远程字典怎么处理

推荐统一使用插件级 `dictApi`，在字段上通过 `dictName` 引用。

## 5. 后端分页字段名不是 `current` / `size`

通过插件级 `tableApiSetting` 适配，不要在每个页面重复转换。

## 6. AI 能直接根据这份文档生成页面吗

可以，前提是输入信息完整，且明确页面结构、接口协议和字段约束。推荐先阅读 [AI 使用建议](/guide/ai-usage)。
